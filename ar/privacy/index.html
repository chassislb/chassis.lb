سياسة الخصوصية

كيف تتعامل Chassis مع المعلومات والاستفسارات.

توضّح سياسة الخصوصية هذه ما هي المعلومات التي قد تجمعها Chassis عبر هذا الموقع، كيف يمكن استخدامها، وكيف قد تدخل المنصات الخارجية في العملية بحسب نوع التفاعل أو المشروع أو الإعداد التقني.

نظرة عامة على السياسة

آخر تحديث: 20 مايو 2026

تجمع Chassis فقط المعلومات اللازمة للتواصل، فهم الاستفسارات، وتحسين تجربة الموقع أو المشروع.

Chassis هي براند مختص بهيكلة الأعمال، أنظمة العمليات، والبنية التنفيذية. قد يجمع هذا الموقع معلومات عندما يتواصل الزوار مع Chassis، يرسلون استفسارات، يحجزون مكالمات، يطلبون خدمات، أو يتفاعلون مع ميزات الموقع.

قد تتم معالجة المعلومات أيضاً عبر منصات خارجية بحسب الموقع، نطاق المشروع، متطلبات العميل، إعدادات الاستضافة، طريقة التواصل، إعداد التحليلات، نظام الدفع، قاعدة البيانات، أو التنفيذ التقني.

Chassis لا تبيع المعلومات الشخصية.

المعلومات التي تغطيها هذه السياسة

بسيطة، عملية، ومرتبطة بهذا الموقع واستفسارات العملاء.

01

معلومات التواصل

الاسم، البريد الإلكتروني، رقم الهاتف، اسم البزنس، تفاصيل المشروع، تفاصيل الاستفسار، أو أي معلومات يتم إرسالها مباشرة عبر النماذج، المكالمات، الرسائل، أو الإيميل.

02

بيانات استخدام الموقع

تحليلات أساسية مثل الصفحات التي تمت زيارتها، نوع الجهاز، المتصفح، الموقع التقريبي، مصدر الزيارة، وطريقة تفاعل الزوار مع الموقع.

03

بيانات المشاريع والجدولة

المعلومات التي تتم مشاركتها عند حجز مكالمة، مناقشة مشروع، طلب عرض، إرسال تفاصيل بزنس، أو التواصل بخصوص الخدمات.

04

بيانات تقنية

معلومات تقنية قد تتم معالجتها عبر الاستضافة، التحليلات، قواعد البيانات، النماذج، الأتمتة، أدوات الأمان، سلوك المتصفح، أو المنصات المرتبطة بالمشاريع.

التفاصيل

كيف يتم التعامل مع المعلومات.

تُستخدم المعلومات التي يتم جمعها للرد، التحسين، التشغيل، وتنفيذ العمل بشكل صحيح.

01

المعلومات التي يتم جمعها

قد تجمع Chassis الأسماء، عناوين البريد الإلكتروني، أرقام الهاتف، أسماء الأعمال، تفاصيل الاستفسارات، تفاصيل المشاريع، تفاصيل حجز المكالمات، بيانات استخدام الموقع، والمعلومات التقنية اللازمة لفهم الطلب أو دعمه.

02

كيف يتم استخدامها

قد تُستخدم المعلومات للرد على الاستفسارات، فهم احتياجات البزنس، جدولة المكالمات، إعداد العروض، تقديم الخدمات، تحسين أداء الموقع، الحفاظ على التواصل، وتقييم الاهتمام بخدمات Chassis.

03

المنصات الخارجية

قد يستخدم هذا الموقع ومشاريع Chassis منصات خارجية للتحليلات، الجدولة، الاستضافة، قواعد البيانات، النماذج، المدفوعات، التواصل، الأتمتة، تخزين الملفات، تسليم المشاريع، أو التنفيذ التقني. قد تختلف المنصات المستخدمة بحسب العميل، المشروع، النطاق، والإعداد المختار.

ملاحظات إضافية

تتخذ Chassis خطوات معقولة لحماية المعلومات المرسلة، لكن لا يمكن ضمان أي إرسال عبر الإنترنت، استضافة، قناة تواصل، قاعدة بيانات، أو تخزين رقمي على أنه آمن بالكامل. يجب على الزوار والعملاء تجنب إرسال معلومات شديدة الحساسية أو سرية إلا إذا كان هناك اتفاق مناسب أو عملية آمنة واضحة.

لا يتم بيع المعلومات لأطراف ثالثة

تُستخدم الاستفسارات المرسلة للتواصل وتقييم الخدمة

قد تُستخدم التحليلات لفهم أداء الموقع

قد تختلف الأدوات والمنصات حسب المشروع واحتياجات العميل

خياراتك

تواصل مع Chassis إذا كانت لديك أسئلة حول الخصوصية.

يمكنك طلب توضيح، تصحيح، أو حذف المعلومات التي أرسلتها مباشرة إلى Chassis.

يمكنك التواصل مع Chassis من أجل

  • السؤال عن المعلومات التي أرسلتها عبر الموقع
  • طلب تصحيح معلومات تواصل غير دقيقة
  • طلب حذف معلومات الاستفسار المباشر حيثما أمكن
  • طرح أسئلة حول سياسة الخصوصية هذه

المنصات الخارجية

  • قد تجمع أدوات التحليلات بيانات استخدام الموقع
  • قد تعالج أدوات الجدولة أو التواصل تفاصيل الحجز أو معلومات التواصل
  • قد تعالج منصات الاستضافة، قواعد البيانات، الدفع، الأتمتة، أو الأدوات التقنية بيانات مرتبطة بالمشروع بحسب الإعداد المختار
  • المنصات الخارجية تتبع سياسات الخصوصية وممارسات البيانات الخاصة بها
